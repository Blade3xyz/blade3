# Blade<sup>3</sup>
Port forwarding tool
![](./demo.png)

## About
Blade3 forwards local services to remote servers over an encrypted tunnel. SSH not required.

## Install
```bash
/usr/bin/env ruby -e "$(curl -fsSL https://install.blade3.xyz/install.rb)"
```
